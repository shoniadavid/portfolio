About

Hello, I’m David,
self-taught front-end developer, currently based in Georgia, Tbilisi –
I create websites with knowledge and passion.

With the experience of fewer than six months of coding, I'm a junior front-end web developer, who is motivated to learn something new every day. I try to express my vision of web design and development at my full potential using modern technologies. I'm also getting familiar with graphic and UI/UX designs, so I can make websites with beautiful and user-friendly interfaces. As you can see from my personal portfolio (I hope so), I very like minimalism in web design.

Besides design, my main goal while doing the project is to write code clean and understandable for my fellow web developers.

I work with the latest techniques in web development and my agile workflow is the result of the precisely combined tools and strategies that include object oriented patterns, build scripts, pre-processors and minification, source code management, testing as well as deployment.

With my knowledge in both design and technologies I make beautiful, fast and responsive websites, which meets the requirements of modern technology and design standarts at the same time.
